## Что такое программирование?

**Программирование** — это процесс создания инструкций для компьютера, которые говорят ему, что делать. Представьте, что вы объясняете ребенку, как приготовить бутерброд:

1. Возьми хлеб
2. Намажь масло
3. Положи колбасу
4. Накрой вторым кусочком хлеба

Аналогично, программирование — это написание пошаговых инструкций для компьютера.

### Что такое алгоритм?

**Алгоритм** — это последовательность действий для решения конкретной задачи. В программировании алгоритмы помогают нам разбить сложную проблему на простые, понятные шаги.

Пример алгоритма "Найти наибольшее число из трех":
1. Сравни первое и второе число
2. Запомни большее из них
3. Сравни запомненное число с третьим
4. Выведи большее

## История и особенности C#

### История создания

**C#** (произносится "Си-Шарп") был создан компанией Microsoft в 2000 году под руководством Андерса Хейлсберга. Язык был разработан как часть платформы .NET Framework.

### Почему именно C#?

- **Простота изучения**: C# сочетает мощь C++ с простотой Visual Basic
- **Объектно-ориентированный**: Помогает организовать код логично и понятно
- **Типобезопасность**: Предотвращает многие ошибки на этапе компиляции
- **Автоматическое управление памятью**: Сборщик мусора освобождает память автоматически
- **Богатая библиотека**: .NET Framework предоставляет тысячи готовых классов и методов

### Где используется C#?

- **Веб-приложения** (ASP.NET)
- **Десктопные приложения** (Windows Forms, WPF)
- **Мобильные приложения** (Xamarin)
- **Игры** (Unity)
- **Облачные сервисы** (Azure)

## .NET Framework и CLR

### .NET Framework

**.NET Framework** — это платформа для разработки и выполнения приложений на Windows. Она включает:

- **Библиотеку базовых классов** (BCL) — тысячи готовых классов
- **Общую языковую среду выполнения** (CLR)
- **Компиляторы** для разных языков (C#, VB.NET, F#)

### CLR (Common Language Runtime)

**CLR** — это "виртуальная машина", которая выполняет ваши программы. Она обеспечивает:

- **Управление памятью** (сборка мусора)
- **Безопасность типов**
- **Обработку исключений**
- **Межъязыковую совместимость**

## Структура программы на C#

Каждая программа на C# имеет определенную структуру. Рассмотрим простейший пример:

```csharp
using System;

namespace MyFirstProgram
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Привет, мир!");
        }
    }
}
```

Разберем по частям:

1. **using System;** — подключает пространство имен System
2. **namespace** — логически группирует классы
3. **class Program** — объявляет класс с именем Program
4. **static void Main** — точка входа в программу
5. **Console.WriteLine** — выводит текст в консоль

## Основные понятия

### Комментарии

**Комментарии** — это текст в коде, который игнорируется компилятором. Они помогают объяснить, что делает код.

```csharp
// Это однострочный комментарий

/* Это
   многострочный
   комментарий */

/// <summary>
/// Это XML-комментарий для документации
/// </summary>
```

### Пространства имен

**Пространство имен** (namespace) помогает организовать классы логическими группами и избежать конфликтов имен.

```csharp
namespace Математика
{
    class Калькулятор
    {
        // Методы для вычислений
    }
}

namespace Геометрия
{
    class Калькулятор
    {
        // Методы для геометрических вычислений
    }
}
```

## Типичные ошибки новичков

### 1. Забыть точку с запятой
```csharp
// НЕПРАВИЛЬНО:
Console.WriteLine("Привет")

// ПРАВИЛЬНО:
Console.WriteLine("Привет");
```

### 2. Неправильный регистр
```csharp
// НЕПРАВИЛЬНО:
console.writeline("Привет");

// ПРАВИЛЬНО:
Console.WriteLine("Привет");
```

### 3. Забыть закрывающие скобки
```csharp
// НЕПРАВИЛЬНО:
if (true)
{
    Console.WriteLine("Да");
// Забыли закрывающую скобку }

// ПРАВИЛЬНО:
if (true)
{
    Console.WriteLine("Да");
}
```
